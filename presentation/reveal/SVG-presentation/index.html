How to connect

  • img-tag 
    <img class="icon" src="img/svg/dots.svg" alt="dots icon">
  • background-image 
    
  <a href="https://facebook.com" class="icon fb"> Facebook </a>
          
  .fb {
    background-image: url(../img/svg/fb.svg);
  }
